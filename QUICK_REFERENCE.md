1. Select mode: **Erase** (red) or **Restore** (green)

3. Draw on image with one finger
4. Strokes auto-apply after 3 strokes

### Using Zoom

2. **Pan**: Two fingers to move around when zoomed

### Controls
- **Apply**: Process strokes manually

- **Smooth**: Refine mask edges
- **Done**: Apply changes and exit
- **Cancel**: Discard changes and exit

## ğŸ¨ Common Tasks

### Remove Background Fragments
```
1. Select Erase mode (red)
2. Brush size: 80-100px
3. Zoom: 1x for large areas
4. Paint over unwanted areas
5. Tap Apply
```

### Restore Missing Parts
```
1. Select Restore mode (green)
2. Brush size: 50px
3. Zoom: 2-3x for precision
4. Paint over removed areas
5. Tap Apply
```

### Fix Hair/Edges
```
1. Zoom: 3-4x
2. Brush size: 15-25px
3. Hardness: 30-40% (in Advanced)
4. Draw carefully along edges
5. Tap Smooth when done
```

## ğŸ› Troubleshooting

### Zoom feels laggy
- Close other apps
- Try smaller image (app auto-resizes to 2048px)

### Strokes not applying
- Wait for auto-apply (after 3 strokes)
- Or tap Apply button manually
- Check for processing indicator

### Can't see details
- Pinch to zoom in (up to 5x)
- Use smaller brush size
- Double-tap for quick 2.5x zoom

### Drawing outside image
- Fixed! Strokes only register on image
- Check you're within image bounds

### Aspect ratio looks wrong
- Fixed! Always maintains correct ratio
- Letterboxing added automatically
